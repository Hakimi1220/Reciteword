# 遗忘曲线功能使用说明

## 功能概述

本次更新为单词背诵模块加入了基于艾宾浩斯遗忘曲线的智能复习系统，主要包含以下三个核心功能：

### 1. 不记得的单词5个后重新弹出

当你在背诵过程中点击"不记得"或"补好>_<，还是不记得"时，该单词会被标记为需要加强复习。在继续背诵5个单词后，系统会自动将这个单词重新插入到学习队列中，帮助你及时巩固。

### 2. 连续三次记住的单词不再推送

如果你连续3次都正确记住了某个单词，系统会认为你已经完全掌握了这个单词，将其标记为"已掌握"状态，后续的背诵中将不再推送该单词，让你的学习更高效。

### 3. 基于遗忘曲线的智能复习调度

系统会根据艾宾浩斯遗忘曲线算法，自动计算每个单词的最佳复习时间：

- **第1次复习**：学习后30分钟
- **第2次复习**：1小时后
- **第3次复习**：8小时后
- **第4次复习**：1天后
- **第5次复习**：3天后
- **后续复习**：7天后

如果你连续正确回答，复习间隔会适当延长；如果回答错误，系统会重置复习进度。

## 使用方式

### 开始背诵

1. 点击"背诵模式"标签
2. 选择显示模式（中文提示或英文提示）
3. 点击"开始背诵"按钮

系统会自动根据遗忘曲线选择需要复习的单词。如果没有到复习时间的单词，则会显示新单词供你学习。

### 背诵过程

- **记得**：点击后会显示完整释义，然后提供两个选项：
  - "我真的记住啦"：标记为正确，进入下一个单词
  - "补好>_<，还是不记得"：标记为错误，该单词会在5个单词后重新出现

- **不记得**：点击后会显示完整释义，然后提供一个选项：
  - "这次一定记住"：标记为错误，该单词会在5个单词后重新出现

### 完成背诵

当你完成所有需要复习的单词后，系统会显示完成界面。如果所有单词都已掌握（连续3次正确），系统会提示"恭喜！所有单词都已掌握，暂无需要复习的单词"。

## 数据存储

所有的学习进度和统计信息都会自动保存在浏览器的本地存储中，即使关闭浏览器也不会丢失。数据包括：

- 单词列表
- 每个单词的学习统计（连续正确次数、复习次数、下次复习时间等）

## 注意事项

1. **首次使用**：如果是第一次使用背诵功能，系统会随机显示所有单词供你学习
2. **复习时机**：建议每天定期打开应用进行复习，让遗忘曲线发挥最大效果
3. **数据备份**：由于数据存储在浏览器本地，建议定期导出重要数据（未来版本会添加导出功能）

## 技术实现

- 使用艾宾浩斯遗忘曲线算法计算复习间隔
- 动态调整学习队列，实时插入需要复习的单词
- 自动过滤已掌握的单词，提高学习效率
- 所有数据持久化存储在本地的localStorage中

## 测试场景

### 测试1：不记得的单词5个后重新弹出

1. 添加至少6个单词到单词本
2. 开始背诵
3. 在第1个单词时点击"不记得" -> "这次一定记住"
4. 继续背诵接下来的5个单词（都点击"记得" -> "我真的记住啦"）
5. 观察第7个位置是否出现了第1个单词

**预期结果**：第7个单词应该是之前标记为"不记得"的第1个单词

### 测试2：连续三次记住的单词不再推送

1. 添加一个单词到单词本
2. 第一次背诵该单词，点击"记得" -> "我真的记住啦"
3. 重新开始背诵，再次遇到该单词，点击"记得" -> "我真的记住啦"
4. 第三次背诵该单词，点击"记得" -> "我真的记住啦"
5. 重新开始背诵

**预期结果**：系统提示"恭喜！所有单词都已掌握，暂无需要复习的单词"

### 测试3：遗忘曲线时间调度

1. 添加几个单词并学习一次
2. 等待30分钟后再次打开应用
3. 点击"开始背诵"

**预期结果**：应该能看到之前学习过的单词出现在复习队列中

### 测试4：混合场景

1. 添加10个单词
2. 第一次学习所有单词，其中3个标记为"不记得"
3. 继续背诵，观察这3个单词是否在5个单词后重新出现
4. 对重新出现的单词，如果连续3次都标记为"记得"
5. 重新开始背诵

**预期结果**：
- 不记得的单词会在5个单词后重新出现
- 连续3次正确的单词不会再出现
- 其他单词根据遗忘曲线时间安排复习
